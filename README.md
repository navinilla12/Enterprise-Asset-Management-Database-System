# Enterprise Asset Management System

## Overview
This project simulates an enterprise-level utility asset management database system.

## Tech Stack
- Python
- SQLite
- Pandas

## Features
- Asset lifecycle tracking
- Maintenance scheduling system
- Inspection failure analysis
- Outage duration analytics
- Relational schema design

3. Run all cells
